Cryptocurrency คืออะไร และคุณควรที่จะลงทุนกับมันหรือไม่


 Cryptocurrency หรือหากแปลเป็นไทยตรงตัวก็คือ “สกุลเงินที่ถูกเข้ารหัส” ซึ่งเป็นการเล่นคำสองคำคือ cryptography และ currency เข้าด้วยกัน นั่นหมายความว่า Bitcoin, Ethereum, Litecoin หรือที่คนบนทีวีพูดกันนั้นเป็นสิ่งที่ไม่มีตัวตน และจับต้องไม่ได้ ซึ่งมันจะทำงานอย่างลับ ๆ ในคอมพิวเตอร์ สิ่งที่จะสามารถมองเห็นได้มีเพียงตัวเลข ลองมาดูการเข้ารหัสและถอดรหัสแบบง่าย ๆ โดยอ้างอิงจากเว็บ md5decrypt.net หากทดสอบนำข้อความคำว่า “สวัสดีครับ” เข้าไป encrypt หรือเข้ารหัสไว้สิ่งที่เราเห็นจะมีลักษณะแบบนี้

 และเช่นเดียวกัน ถ้าหากนำเอาชุดตัวเลขดังกล่าวไปถอดรหัสบนเว็บดังกล่าว ก็จะได้กลับมาเป็นคำว่า “สวัสดี” ซึ่งหากคุณเพิ่มตัวอักษรเข้ามา 1 ตัว หรือทำการลดตัวอักษรหนึ่งตัวละก็ ชุดตัวเลขและตัวหนังสือเหล่านั้นก็จะถูกเปลี่ยนใหม่หมด แต่ถ้าเป็น Public key และ Private key จะมีแค่การเข้าและถอดรหัสแบบง่าย ๆ เราคงใช้ Cryptocurrency กันไม่ได้แน่นอน โชคดีที่เรามีการเข้ารหัสแบบ public-key cryptography หรือก็คือการเข้าและถอดรหัสแบบใช้กุญแจ กุญแจที่ว่านี้จะมีอยู่สองแบบ คือ public key และ private key โดยลักษณะหน้าตาของ public key จะเป็นแบบนี้

 สมมติว่าตู้ล็อคนี้เป็นของต้นไม้ และล็อคนั้นจะมีรูปแบบในการล็อคสามขั้นตอนคือ A (ล็อค), B (ปลดล็อค) และ C (ล็อค) และกุญแจสำหรับตู้ล็อคนี้จะมีอยู่สองดอกคือ ดอกแรกที่ใช้หมุนตามเข็มนาฬิกา (จาก A ไป B ไป C) และดอกที่สองที่ใช้หมุนทวนเข็มนาฬิกา (จาก C ไป B ไป A) ต้นไม้หยิบกุญแจดอกแรก แล้วเก็บไว้ที่ตัวเอง กุญแจดอกนี้เราจะเรียกว่า “private key” เพราะเป็นกุญแจส่วนตัวที่มีเพียงต้นไม้คนเดียวเท่านั้นที่มีมัน ส่วนกุญแจดอกที่สองจะถูกเรียกว่า “public key” ซึ่งต้นไม้จะทำการก็อปปี้กุญแจดอกนี้ไว้นับร้อย หรือเท่าไรก็ได้ตามที่เขาต้องการ และแจกมันให้กับเพื่อนของเขา และครอบครัวของเขา บางทีเขาก็วางมันไว้บนโต๊ะของเขา, ในออฟฟิศของเขา หรือแขวนมันไว้ที่ประตูหน้าบ้าน และถ้ามีใครถามหานามบัตรของต้นไม้ เขาก็จะเอากุญแจดอกนี้ให้ไปด้วยพร้อมๆกัน แล้วถ้า ต้นไม้มี private key ที่สามารถหมุนจากจุด A ไป B ไป C ในขณะที่เพื่อนคนรู้จักของต้นไม้ทุกคนถือ public key ที่สามารถหมุนจาก C ไป B ไป A ได้ ลองคิดต่อดูว่าถ้าเกิดคุณต้องการส่งเงินให้ต้นไม้เป็นค่าจ้างปั่นราคา คุณก็สามารใส่เงินลงในตู้เซฟ และใช้กุญแจ public key ที่ก็อปมานั้นทำการล็อค แล้วจำไว้ให้ดีว่ากุญแจ public key ของต้นไม้นั้นหมุนได้ทวนเข็มนาฬิกาเท่านั้น ดังนั้นเมื่อคุณใส่เงินเข้าไปแล้ว คุณก็จะต้องหมุนกุญแจไปตำแหน่ง A ซึ่งในขณะนี้ตู้เซฟนั้นได้ถูกล็อคไว้เรียบร้อยแล้ว และมีเพียงกุญแจดอกเดียวเท่านั้นที่สามารถหมุนจากตำแหน่ง A ไป B ได้ ซึ่งคือ private key ของต้นไม้เท่านั้น สรุปคือ ใครที่มี public key ของต้นไม้จะสามารถส่งอะไรก็ได้ให้ไปในตู้เซฟของเขา, ล็อคมัน และคนที่จะทำการเปิดมันได้จะมีเพียงแค่เจ้าของตู้เซฟและ private key นั้น ซึ่งก็คือต้นไม้ แต่ยังมีกรณีการใช้งานแบบพิเศษสำหรับตู้เซฟที่ว่านี้อยู่ ลองคิดดูว่าถ้าต้นไม้ใส่เอกสารสำคัญเข้าไปในตู้เซฟนั้นด้วยตัวเอง และเขาใช้ private key ของเขาทำการล็อคตู้นั้น และหมุนไปที่ตำแหน่ง C คำถามคือ เขาจะทำแบบนั้นทำไม เพราะว่าใครก็ตามที่มี public key จะสามารถปลดล็อคมันได้หมด ลองคิดต่อว่าเมื่อมีคนส่งตู้เซฟนี้มาให้คุณ และบอกว่ามันเป็นของต้นไม้ แต่คุณไม่เชื่อเขา คุณจึงทดสอบหยิบ private key ของต้นไม้ที่คุณเคยเก็บไว้มาลองไขดู และลองหมุนไปด้านขวา ผลคือไม่มีอะไรเกิดขึ้น คุณเลยลองหมุนไปด้านซ้าย ตู้เซฟนี้เปิดออก ตอนนี้คุณเชื่อแล้วว่าตู้เซฟนี้เป็นของต้นไม้จริง เพราะสิ่งเดียวที่หมายถึงคือตู้เซฟที่ถูกล็อคโดยต้นไม้คนเดียวเท่านั้น ตอนนี้คุณรู้แล้วว่ามีเพียงต้นไม้ที่ใส่เอกสารลงไปในตู้เซฟนี้ ไม่ใช่ใครอื่นที่ไหน โดยเราเรียกสิ่งนี้ว่า “digital signature” หรือลายเซ็นดิจิตอลนั่นเอง สรุปคือเมื่อคุณทำการเข้ารหัส (ล็อค) ด้วย public key มีเพียงคนที่ถือ private key เท่านั้นที่สามารถปลดล็อคมันได้ เช่นเดียวกัน ถ้าคุณเข้ารหัส (ล็อค) มันด้วย private key ใครก็ตามที่มี public key จะไม่สามารถถอดรหัส (ปลดล็อค) มันได้ แต่วิธีการดังกล่าวนั้นถูกใช้เพื่อเป็นการยืนยันว่าตู้เซฟดังกล่าวนั้นถูกเซ็นและยืนยันด้วยตัวของคุณเองเท่านั้น แล้วเกี่ยวกับ Cryptocurrency ตรงไหน? ลองคิดตามว่าถ้าคุณซื้อกาแฟหนึ่งแก้วด้วยเงินสดธรรมดา สิ่งที่ต้องทำแค่ควักเงินในกระเป๋าของคุณที่มีตัวเลขบอกมูลค่าของมันอยู่ (ว่ามีกี่บาท) และยื่นมันให้คนขายกาแฟ บาริสตาถึงยื่นกาแฟส่งมาให้คุณแล้วรับเงินของคุณไป ปัญหาของเงินสดในปัจจุบันคือ มันถูกเปลี่ยนมือเป็นร้อยเป็นพันครั้ง และไม่มีใครสามารถรู้ได้ว่าธนบัตรในมือพวกเขานั้นเคยผ่านใครมาบ้าง ซึ่งทำให้การติดตามและค้นหาเส้นทางการทำธุรกรรมสำหรับเงินสดเป็นเรื่องที่ยากมาก สกุลเงินดิจิตอลจึงถูกคิดค้นเพื่อมาแก้ปัญหานี้ ความแตกต่างคือ สกุลเงินดิจิตอลถูกสร้างขึ้นในรูปแบบดิจิตอลเท่านั้น ไม่สามารถจับต้องได้ ลองคิดดูว่าเราทำการสแกนธนบัตรมูลค่าห้าร้อยบาทออกมาเป็นไฟล์คอมพิวเตอร์ แล้วตั้งชื่อมันว่า ‘five-euros.jpg’ ซึ่งเมื่อมันถูกแปลงเป็นเงินดิจิตอลแล้วมันจะมีข้อได้เปรียบมากมาย แต่ก็มีผลเสียมาด้วยเช่นกัน เช่นไฟล์เงินดิจิตอลเหล่านี้สามารถถูก copy และ paste ได้ไม่จำกัด อีกทั้งยังไม่สามารถแยกความแตกต่างระหว่างเงินของจริงและเงินดิจิตอลได้ แล้วถ้าคุณเป็นคนขายกาแฟ และลูกค้าทำการ copy ไฟล์เงิน ‘five-euros.jpg’ มาจ่ายคุณ และยังนำมันไปจ่ายที่ร้านอื่นด้วย เพราะไฟล์สองไฟล์ที่เหมือนกันทุกอย่างทำให้ไม่สามารถแยกได้ว่าใครเป็นเจ้าของที่แท้จริงของไฟล์เงินดังกล่าว ข้อเสียอีกหนึ่งข้อคือ เมื่อมันถูก copy และ paste ได้ไม่จำกัด นั่นหมายความว่าผู้ที่ทำการ copy นั้นจะสามารถจ่ายเงินกี่ครั้งก็ได้ตามที่พวกเขาต้องการ Cryptocurrency จึงถูกคิดค้นมาเพื่อแก้ปัญหาดังกล่าวนี้

 เมื่อผู้รับเงินต้องการจะตรวจสอบว่าไฟล์เงิน ‘five-euros.jpg’ นี้เป็นของใคร เขาสามารถที่จะรอให้ต้นทางเซ็นชื่อของเขาโดยใช้ private key และส่งมันไปพร้อม ๆ กับไฟล์ดังกล่าว สองอย่างนี้รวมกันคือ digital token และเมื่อปลายทางได้รับ token ดังกล่าวแล้วก็สามารถตรวจสอบมันได้ด้วย public key ของเขา เพื่อยืนยันว่า five-euros.jpg ตัวนี้เป็นของใคร ธุรกรรมครั้งนี้นับเป็นหนึ่งธุรกรรม แต่สำหรับ Bitcoin หรือ cryptocurrency มันมีมากกว่านั้น เมื่อธุรกรรมทั้งหมดนี้ถูกจัดเก็บเอาไว้ในสมุดบัญชี สิ่งหนึ่งที่ทำให้ระบบการจัดการด้าน Internet banking หรือการทำธุรกรรมในปัจจุบันผ่านอินเทอร์เนตอยู่ได้นั้นคือ ledger หรือสมุดบัญชีที่เอาไว้ใช้บันทึกข้อมูลการทำธุรกรรมทั้งหมด หากปราศจากซึ่งสมุดบัญชีแล้ว เงินเหล่านั้นก็คงไม่ต่างจาก ‘five-euros.jpg’ ที่ copy และ paste และส่งหากันอย่างไร้การควบคุม ทำให้ cryptocurrency หรือ Bitcoin นั้นแตกต่างจากสกุลเงินดิจิตอลธรรมดาทั่วไป นั้นคือการบันทึกธุรกรรมของมันนั้นถูกเก็บไว้บนสมุดบัญชีที่เรียกว่า Blockchain Bitcoin เป็นสกุลเงินดิจิตอลที่ใช้เทคนิคการเข้าและถอดรหัสในรูปแบบด้านบน เมื่อตัวเลขที่ไม่สามารถถูกจับต้องได้ถูกส่งจากบุคคลหนึ่งสู่อีกบุคคลหนึ่ง สมุดบัญชีที่ว่านี้จะทำการเก็บข้อมูลธุรกรรมที่ว่านี้ไว้ หากคุณอยากจะเปิดบัญชีธนาคารสักแห่งหนึ่ง สิ่งที่คุณต้องทำก็แค่เดินเข้าธนาคารที่คุณชอบ และกำเงินเข้าไปขอเปิดบัญชี สิ่งที่คุณจะได้รับกลับมาก็คือสมุดบัญชีหนึ่งเล่ม ที่มีเลขบัญชีและชื่อของคุณสลักไว้ แต่สำหรับ Bitcoin นั้น ทุกคนสามารถสร้าง public และ private key ที่สามารถถูกใช้ได้โดยเปรียบเสมือนบัญชีธนาคารที่เอาไว้รับเงิน โดย public key ที่ว่านี้เราจะเรียกมันว่า ‘Bitcoin address’ ซึ่งของแต่ละคนจะไม่มีวันเหมือนกันและซ้ำกันได้เมื่อถูกระบบสร้างมันออกมา ดังนั้นหากจะให้เปรียบคือ address ของ Bitcoin นั้นไม่ได้ต่างจากเลขบัญชีธนาคาร เพียงแต่มันไม่ได้ถูกผูกกับธนาคารใดไว้ และไม่มีใครมาเป็นเจ้าของมัน ถ้าลองทำธุรกรรม Bitcoin ดู เพื่อให้แน่ใจว่าผู้ทำธุรกรรมเป็นเจ้าของ Bitcoin จริง การทำธุรกรรมจะต้องถูกเซ็นกำกับด้วย private key หรือเจ้าของ Bitcoin address ตัวนั้น หลังจากที่ลายเซ็นของผู้ทำธุรกรรมรวมถึง public key ของเขาจะถูกแทรกไว้ในธุรกรรม ซึ่งสามารถทำให้ทุกคนเข้ามาตรวจสอบธุรกรรมของ Bitcoin ได้ว่าถูกส่งมาจากเจ้าของตัวจริง ที่มี Bitcoin จำนวนนั้นจริง แม้ทุกคนจะมี public key ของผู้ทำธุรกรรมหมดทุกคน แต่ในเชิงทฤษฎีแล้ว ไม่มีใครสามารถรู้ได้เลยว่าตัวตนของเจ้าของ public key นั้นคือใคร เช่นกันกับ Bitcoin ปัจจุบันมี address นับล้านล้าน address แต่ไม่มีใครสามารถรู้ได้เลยว่า address นี้เป็นของใคร กระนั้น การยกตัวอย่าง ระหว่างคนสองคนมาเจอหน้ากัน ซึ่งหากเกิดขึ้นได้ในชีวิตจริง เราคงทราบดีว่าได้รับ Bitcoin มาจากคน ๆ นี้ เพียงแต่ความซับซ้อนจะเริ่มขึ้นหลังจากนั้น เนื่องจาก cryptocurrency แตกต่างจากสกุลเงินดิจิตอลแบบเก่า ทำให้ธุรกรรมที่เกิดขึ้นต่อจากนี้จะไม่ถือเป็นการสร้างเงินขึ้นมาใหม่ แต่จะเป็นการถูกนำไปส่งทำธุรกรรมต่อ ความได้เปรียบของ Bitcoin หรือ cryptocurrency คือ Blockchain ที่ทำหน้าที่เป็นสมุดบัญชีเก็บธุรกรรมและเก็บหลักฐานการทำธุรกรรมไว้หมดตั้งแต่อดีตปฐมกาลจนถึงปัจจุบัน นั่นหมายความว่าผู้ที่ใช้งานสามารถจะตรวจสอบหาแหล่งที่มาได้ไปจนถึงจุดเริ่มต้นของมันเลย หลายครั้งที่การบันทึกการทำธุรกรรมนั้นค่อนข้างจะซับซ้อน ซึ่งจำนวนทศนิยมของ Bitcoin นั้นสามารถถูกแบ่งแยกย่อยได้เยอะมาก นั่นหมายความว่า ผู้ที่ซื้อ Bitcoin ไม่จำเป็นต้องบังคับตัวเองให้ซื้อ 1 BTC เสมอไป แต่คุณสามารถที่จะเลือกซื้อมันเป็นจำนวนเล็กน้อยอย่าง 0.00001356 BTC ได้
